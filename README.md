# 중앙대학교 2021-2학기 캡스톤디자인

## 프로젝트 소개
- ### 주제: 홈 트레이닝 자세 분석 서비스

- ### 세부내용
  - 캠을 이용하여 본인의 운동모습을 촬영하면, 웹 사이트에서 자세에 대한 피드백을 제공
  - 본인의 운동기록을 캘린더 형식으로 볼 수 있음.
  - TensorFlow의 PoseNet을 사용해 관절포인트 획득 + Object-Segmentation을 통한 사진배경 제거 + OpenCV 이미지 처리 ⇒ 스쿼트 운동 시 자세 분석.
